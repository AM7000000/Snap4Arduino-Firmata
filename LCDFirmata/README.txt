1) Push the LCDFirmata.ino sketch into your Arduino board by using the Arduino IDE
2) Import the LCD library into Snap4Arduino by clicking on the file menu and selecting "Import...", then choosing the LCD-library.xml file
3) Follow the instructions in http://arduino.cc/en/pmwiki.php?n=Tutorial/LiquidCrystal (look for the Circuit section) to wire your LCD display
